## Installation

Make sure you have NERDTree installed locally

#### VimPlug

Add the configuration below after the NERDTree plugin in your vim configuration:

```vim
Plug 'paulondc/vim-nerdtree-open-externally'
```

Then run the following in Vim:

```vim
:source %
:PlugInstall
```
